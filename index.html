🔗 Kết nối qua Google Apps Script

Chỉ cần 1 URL Web App — dùng cho cả đọc lẫn ghi. Không cần API Key hay Spreadsheet ID riêng.

📖 Hướng dẫn Setup (3 bước)

BƯỚC 1 — Tạo Google Spreadsheet
Vào sheets.google.com → Tạo file mới.
Đặt tên file tùy ý (VD: BHX Biệt Kích DB). Giữ nguyên, không cần tạo thêm sheet con.

BƯỚC 2 — Tạo Apps Script & Deploy
Trong Spreadsheet → menu Tiện ích mở rộng (Extensions) → Apps Script
Xóa toàn bộ code cũ → nhấn nút bên dưới để copy code mới → dán vào → Lưu (Ctrl+S)
Nhấn Triển khai (Deploy) → Triển khai mới → Web App
  • Thực thi với tư cách: Tôi (Me)
  • Ai có quyền truy cập: Bất kỳ ai (Anyone)
→ Triển khai → Cấp quyền nếu được hỏi → Copy URL Web App

BƯỚC 3 — Dán URL vào ô bên trên & Lưu
Dán URL vừa copy vào ô "Web App URL" → nhấn "Lưu cấu hình" → "Test kết nối".
Khi test thành công, nhấn "Đẩy Master Data lên Sheets" để đẩy dữ liệu lên.
